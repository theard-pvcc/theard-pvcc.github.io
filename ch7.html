Briefly describe the five layers of TCP/IP (The Internet Protocol hierarchy):

  1. Physical layer
    • Govern the exchange of binary digits across a physical communication channel
  2. Data link layer
    • Addresses and solves error handling and framing. This is done through two stages called Layer 2a, Medium access control, and Layer 2b, Logical Link Control.
  3. Network layer
    • Delivers a message from its sight of creation to its ultimate destination.
  4. Transport layer
    • Assigns and remembers port numbers to go with programs. Creates a "program-to-program" delivery service.
  5. Application layer
    • Implements the end user services provided by a network
