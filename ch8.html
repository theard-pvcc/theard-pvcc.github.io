Briefly describe or define:

  1. Information Security
    • Protecting data from those who do not have permission to access it.
  2. authentication
    • Process of verifying that you are the person who has the right to access a particular devices.
  3. authorization
    • Governs what an authenticated user is allowed to do.
  4. dual authentication
    • Adds a second form of verification by sending a one-time code or using another second verification method.
  5. social engineering
    • The process of using people to get the information you want.
  6. virus
    • A computer program that infects host computers and then spreads.
  7. worm
    • Similar to a virus, but can send copies of itself to other nodes on a computer network without having to be carried by an infected host file.
  8. Trojan horse
    • Program that does some legitimate task, but also secretly carries some other form of malicious program.
  9. ransomware
    • Invades the user's computer, often by email, and then encrypts files on that computer, and holds them ransome.
  10. phishing
    • Not a direct attack on a computer, but rather a scammer posing as a business to attempt to gain access to confidential info.
  11. White Hat
    • Security experts trying to keep us safe.
  12. Black Hat
    • The "Bad Guys" trying to gain access to computers and cause damage.
  13. encryption/decryption
    • Encryption: make information meaningless even if it is stolen. Using an algorithm, information is converted into a representation that cannot be understood without the appropriate decryption
    • Decryption: The inverse of encryption, an algorith convers ciphertext back into plain text.
  14. cryptography
    • The science of "secret writing." Encoding plaintext to hide its true meaning.
  15. symmetric encryption algorithm
    • Requires the use of a secret key known to the sender and receiver.
  16. Caesar cipher
    • Also known as a shift cipher, it involves shifting each character in the message to another character some fixed distance farther in the alphabet.
  17. block cipher
    • A group or block of plaintext letters gets encoded into a block ciphertext.
  18. Data Encryption Standard (DES)
    • Encryption algorithm developed by IBM for the US National Bureau of Standards.
  19. public key system
    • The encryption key is broadcast to everyone, but only the receiver has the decryption.
  20. embedded computers
    • Computational devices such as chips, processors, or computers that are embedded within another system.

Briefly describe how a text file can be hidden in a steganographic image:

  • A text file can be hidden in an image by changing the least significant bit of each byte of the image file to match the binary form of the characters in the text.

List six ways to protect yourself from most cyber attacks (see page 406)

  1. Be sure your computer has up-to-date anivirus software.
  2. Be sure your computer has up-to-date firewall software.
  3. Be sure your computer has up-to-date antispyware.
  4. Set your browser to use a pop-up blocker.
  5. Always install the latest security patches.
  6. Don't send personal or financial info in response to any email.
